# Oplentia

Oplentia is a platform that connects entrepreneurs with stakeholders and funders based on the reach and quality of their ideas.

## Features

- AI-driven idea evaluation
- Stakeholder matching
- Micro-investment ecosystem
- Responsive web design

## Technologies

- Java 17
- Spring Boot
- Thymeleaf
- Bootstrap 5

## Getting Started

1. Ensure Java 17 and Maven are installed.
2. Clone the repository.
3. Run `mvn spring-boot:run` to start the application.
4. Open http://localhost:8080 in your browser.

## Pages

- Home: Overview and features
- About: Mission and story
- Discover Startups: Browse ideas
- Startup Idea: Detailed view
- Blog: Articles
- Blog Post: Individual post
